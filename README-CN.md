## 安装使用

- 获取项目代码

```bash
git clone https://github.com.cnpmjs.org/dgiot/dgiot_dashboard.git
```

- 安装依赖

```bash
cd dgiot_dashboard

yarn install
```

- 运行

```bash
yarn dev
```
